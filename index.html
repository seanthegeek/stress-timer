Stress Timer

Download this project as a .zip file Download this project as a tar.gz file

A HTML5-based timer that is designed to act as a social stressor during an exam. A black circle represents the time allotted to complete the exam. The sound of a ticking clock is used to emphasize the passage of time. As time elapses, the circle becomes an increasingly smaller arc (representing the remaining time), and the ticking volume is gradually increased.

A demo can be found here. For best results use Google Chrome.

Dependencies

All dependencies are included, under the terms of their respective licenses.

  • HTML5 Boilerplate
    • jQuery
    • html5shiv (optional, HTML5 IE enabling script)
  • flot (with the pie plugin)
    • ExplorerCanvas (optional, for IE <= 8 support)
  • PreloadJS
  • SoundJS (with Flash plugin for browsers that lack HTML5 audio support)
